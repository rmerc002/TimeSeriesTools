SKIMP

SKIMP facilitates the calculation and visualization of the MAGIC Matrix Profile, an all subsequence length matrix profile. This document explains how to use this tool's features.

Required Files:

  • SKIMP.m (Main file)
  • mpx.m (Subprocess of SKIMP, may be changed out for a faster calculation of the Matrix Profile)
  • muinv.m (Subprocess of mpx)
  • GenerateTimeSeries (Only required for toy example)

Example #1 - Toy Dataset

The following example runs SKIMP on a random walk with a pair of embedded sine waves. We can see two sine waves PATTERN_A (red) and PATTERN_B (green) embedded in a random walk with subsequence lengths 100 and and 25 respectively.
rng(0); % Initialize the seed for random number generation in this script, be sure to execute rng('default') when done
PATTERN_A = sin(2*(2*pi/100) * transpose(0:100)); % Long sine wave
PATTERN_B = sin((2*pi/(100/4)) * transpose(0:(100/4))); % Short sine wave
CLASSES = [0 1 2 0 1 0 2 0]; % 0 Corresponds to noise while 1 and 2 corresponds to PATTERN_A and PATTERN_B.
T = GenerateTimeSeries({PATTERN_A, PATTERN_B}, CLASSES, 'ShowPlot', true, 'Moving', true);
The triangles below correspond with the red and green subsequences above. Unfortunately, the live editor cannot display animations. The y-axis values have no current meaning (could not remove them from the Live Script). The red line indicates the most recently computer row of the Magic Matrix Profile.
MAGICMatrixProfile1 = SKIMP(T, 'ShowImage', true);

SKIMP Options

Skimp has a variety of optional arguments and parameters.

Range

The range specfies the specific subsequence lengths you wish to explore. This value is defaulted to 1:1:length(Dataset)/4. The subsequence lengths must be positive integral values, cannot be larger than half the length of the dataset and must be increasing.
For example, the following explores every other subsequence length between 1 and 250.
MAGICMatrixProfile2 = SKIMP(T,[1:2:120], 'ShowImage', true);

ShowImage

If you do not wish to view (or spend time performing) live updates of the Magic Matrix Profile, ignore the ShowImage parameter (i.e. do not set it to true).
MAGICMatrixProfile3 = SKIMP(T,[1:2:120], 'ShowImage', false);

SaveImage

Lastly, this option saves a "GIF" to your local directory using a randomized 64-character file name. This option is ideal if you do not want to view the animation of the Magic Matrix Profile live but would like to view it at a later time.
MAGICMatrixProfile4 = SKIMP(T, 'SaveImage', true);
FILENAME = 'cdeqobystbwyzwunekdayhhimqavowjlberixdzoszhkmtvcejbniefxrmxcttoe'
